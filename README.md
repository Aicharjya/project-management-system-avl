# Project Management System using AVL Tree

This project is a web-based project management system that uses an AVL tree and sorting algorithms to efficiently manage and organize projects. It allows users to add, remove, and sort projects while maintaining balanced data structures.

## Features
- Add, delete, and manage projects
- AVL tree implementation for efficient data handling
- Sorting using algorithms like QuickSort
- CRUD operations with database integration
- Organized and structured project display

## Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## Purpose
The purpose of this project is to demonstrate the use of data structures (AVL tree) and algorithms in real-world applications like project management systems.
