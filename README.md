# Project Management System using AVL Tree

This project is a full-stack web-based project management system that uses an AVL tree and sorting algorithms to efficiently manage and organize projects. It includes a PHP-based backend API and a MySQL database for data storage and operations.

## Features
- Add, delete, and manage projects
- AVL tree implementation for efficient data handling
- Sorting using algorithms like QuickSort
- User authentication (login, register, logout)
- Backend API using PHP
- Database integration with MySQL

## Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP (Backend API)
- MySQL (Database)

## Project Structure
- `/api` → PHP backend (handles authentication, project logic, and database operations)
  - `/auth` → login, logout, register functionality
  - `/examples` → sample AVL client usage
  - `projects_avl.php` → AVL tree implementation
- Frontend → HTML, CSS, JavaScript files

## Purpose
The purpose of this project is to demonstrate the practical use of data structures (AVL tree) and sorting algorithms in a real-world application like a project management system.
